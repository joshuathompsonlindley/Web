mintop

An incredibly minimal terminal monitoring application written in Go. It was made because I just wanted a small status indicator that shows me CPU, RAM and Battery usage without distracting graphs and colours.

I've only tested it on Linux (Fedora 34), it probably works on macOS and Windows as it uses distatus/battery and shirou/gopsutil but that's not a promise.

It is currently Open Source and available on my GitHub.
