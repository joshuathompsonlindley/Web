Lecturate

Lecturate is an application that allows students to give feedback on university lectures, directly to lecturers.

The project took place over the course of 6 months, with a full paper and poster presentation detailing the results of research, designs, development, and testing.

At the end of the project, 3 code deliverables were produced:

  1. A C# API and MySQL Database (named Fuse)
  2. A C# Web Application for Lecturers (name Spark)
  3. A Java Android Application for Students (named Neon)

Overall, I was awarded a grade of 71 for report, poster and code.
